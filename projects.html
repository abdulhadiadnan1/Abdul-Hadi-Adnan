3. Blogging Website (Content Management System)

For this project, I created a blogging platform where users can write, edit, and publish blog posts. The site allows multiple users to create accounts, manage their posts, and interact with others by commenting on blog posts. The technologies used include:

  • Frontend: HTML, CSS, and Bootstrap for styling, with JavaScript (Vue.js) for dynamic content rendering.
  • Backend: PHP and Laravel for handling user registration, authentication, and managing content in the database.
  • Database: MySQL for storing blog posts, user profiles, and comments.

Features:

  • User authentication (register, login, logout).
  • Create, edit, and delete posts with WYSIWYG (What You See Is What You Get) text editors.
  • A rich commenting system allowing users to comment on blog posts and interact with each other.
  • Search functionality to help users find posts based on keywords and categories.

This project strengthened my knowledge of content management systems, user authentication, and database design while also focusing on a user-friendly interface and interactive user experience.

3. E-Commerce Store (Full-Stack Web Development Project)

This e-commerce project involved building a full-featured online store that enables users to browse products, add items to a shopping cart, and complete a purchase using payment gateways. The project was built using:

  • Frontend: HTML, CSS, JavaScript (React.js) for the user interface. Responsive design was applied to make the site mobile-friendly.
  • Backend: Node.js and Express for handling server requests, user authentication, and database interactions.
  • Database: MongoDB to store product listings, user profiles, order history, and payment details securely.
  • Payment Integration: Integrated Stripe API for processing secure online payments, with features such as order confirmation emails and payment receipts.
  • Admin Dashboard: Implemented a back-office dashboard for managing inventory, viewing sales analytics, and tracking order statuses.

This project provided valuable experience in full-stack development, API integration, and managing user authentication and secure payment systems.
